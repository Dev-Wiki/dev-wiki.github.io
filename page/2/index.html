发表于 2015-07-08   |  

从Futurice公司Android开发者中学到的经验。

摘要

  • 使用 Gradle 和它推荐的工程结构
  • 把密码和敏感数据放在gradle.properties
  • 不要自己写 HTTP 客户端,使用Volley或OkHttp库
  • 使用Jackson库解析JSON数据
  • 避免使用Guava同时使用一些类库来避免65k method limit（一个Android程序中最多能执行65536个方法）
